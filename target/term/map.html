Mapping Support Tool to SNOMED CT (9 Mar 2020 including SARS-CoV-2 codes) by InfoClinic





[Usage] Check "Semantic Tag" -> Type "Term" which you want to map [and Enter] -> The Closest terms of SNOMED CT will be displayed (Up to 40 terms)

[Note 1] Available browser : Chrome, Safari

[Note 2 to Chrome User] if you got a CORS error, you need to install Chrome extension ("Allow-Control-Allow-Origin:*") at https://chrome.google.com/webstore/category/extensions
